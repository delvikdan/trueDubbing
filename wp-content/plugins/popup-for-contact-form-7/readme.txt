This plugin will show the popup when Contact Form 7 has been submitted. You can customize popup title, message and colors from backend.
== Description ==

Popup for Contact Form 7 plugin will display fully customizable and beautiful popup when form is submitted.

Contact Form 7 Popup features: 

* Shows popup when form submitted succesfully.
* You can customize popup content, beackground and font color.
* Popup is fully responsive and compatible in all major browser.

== Installation ==

1. Install and Activate the plugin through the 'Plugins' menu in WordPress
2. You can change settings inside "Contact >> Popup" in wordpress admin. 

== Frequently Asked Questions ==

= Can I customize Popup design? =

Yes, You can customize popup design as per your need in Contact >> Popup settings
